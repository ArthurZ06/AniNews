# AniNews

Real-time Discord bot for tracking anime news and ranking updates.

## Features

- 📰 Automatic anime news updates from IntoxiAnime
- 🔄 RSS feed monitoring every 30 minutes
- 🚫 Duplicate post prevention using local JSON storage
- 🏆 Top anime ranking tracking powered by Jikan API
- 📈 Detects ranking changes and new entries
- 🤖 Built with discord.py

## Tech Stack

- discord.py
- feedparser
- BeautifulSoup4
- Jikan API
